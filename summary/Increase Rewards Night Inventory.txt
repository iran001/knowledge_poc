Manage Rewards Night Inventory Overview - Rewards Night使用IVANI费率类别（GRS系统） - 对应V1费率代码 - 需确保至少5%库存分配给IVANI PART I: Update Rate Code V1 Allocation 1. 登录grs.ihg.com → 点击菜单Pricing → Rate Codes 2. 精确搜索V1 → 点击进入详情页 → 点击菜单选择Allocations 3. 调整分配： - 点击Edit → 选择日期范围/10天模式 - 修改Allocation字段数值 - 保存后关闭 PART II: Configure IVANI At Least Allocation 1. 菜单选择Pricing → Rate Categories 2. 精确搜索IVANI → 点击详情页底部Allocation Rules → At Least/At Most Setup 3. 设置最低分配： - 点击Edit → 指定日期范围 - 在At Least字段填写库存比例（需≥5%） - 保存设置 PART III: Add Flow Through Room Type to V1 操作优势： - 无需手动输入房型数量 - 可灵活调整IVANI预订对应的房型 操作步骤： 1. 确认流量房型： - 菜单选择Inventory → Sleeping Rooms → 记录Flow Through房型代码 2. 绑定到V1费率： - 菜单选择Pricing → Rate Code → 搜索V1进入详情 - 点击Edit → 编辑Price Point → 点击+号添加房型 - 在"Select..."字段输入流量房型代码 - 系统自动刷新后点击Apply Price Point → 保存配置 关键参数： - IVANI/V1对应关系不可更改 - 最低分配阈值5%为硬性要求 - 流量房型需在Sleeping Rooms模块预配置 - 日期范围设置需覆盖目标销售周期 系统操作要点： - 所有修改需通过Save & Close确认 - 分配值调整需提前10天生效 - 房型代码必须为GRS系统有效值 （注：文档中重复的"Proprietary and Confidential"标识已去除）
